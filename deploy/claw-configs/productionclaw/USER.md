## 用户操作的系统

**产品**：深信服 HCI（超融合基础设施）  
**常见版本**：aCloud 6.x / 超融合一体机系列  
**典型规模**：3-10 节点集群  

**常见故障分类**：

| 类型 | 常见症状 |
|---|---|
| **VM 故障** | 虚拟机无法启动、假死、网络不通 |
| **存储故障** | 磁盘报错、IO 超时、数据盘掉线、存储集群降级 |
| **网络故障** | 虚拟机网络不通、vSwitch 配置异常、物理网络故障 |
| **集群故障** | 节点离线、资源调度失败、高可用切换异常 |
| **资源不足** | CPU/内存/磁盘耗尽、迁移失败 |
| **升级问题** | 升级后服务异常、版本兼容性问题 |

**常用诊断命令参考**：

```bash
# 虚拟机状态
virsh list --all
virsh domstate {vm_name}

# 存储状态
ceph status                         # Ceph 存储集群
ceph df                             # 存储使用量
ceph osd tree                       # OSD 拓扑

# 节点状态
kubectl get nodes -n hci            # K8s 节点
systemctl status  {service}        # 服务状态

# 日志
journalctl -u {service} --since "1 hour ago"
tail -f /var/log/acloud/{component}.log
```
