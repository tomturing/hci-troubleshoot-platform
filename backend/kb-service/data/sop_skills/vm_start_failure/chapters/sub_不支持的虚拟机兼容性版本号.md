### 不支持的虚拟机兼容性版本号

#### 现象描述
启动虚拟机失败；报错：不支持的虚拟机兼容性版本号！错误码：0x01002E59


#### 未升级兼容性

##### 判断方法
1、获取启动虚拟机失败任务描述
- acli task get -k "不支持的虚拟机兼容性版本号" -t ${YYYY-MM-DD} -s -1
acli task get -k "不支持的虚拟机兼容性版本号" -t ${YYYY-MM-DD} -s -1
acli task get -k "不支持的虚拟机兼容性版本号" -t ${YYYY-MM-DD} -s -1

##### 解决方案
虚拟机详情界面，更多操作里升级虚拟机兼容性处理
