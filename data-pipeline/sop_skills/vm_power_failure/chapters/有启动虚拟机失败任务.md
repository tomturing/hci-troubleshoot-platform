## 有启动虚拟机失败任务

### 前置检查
获取【启动虚拟机】失败任务的【描述】信息，并根据匹配的关键字到对应章节进行检查
```bash
acli task get -v ${vmid} -t ${YYYY-mm-dd} -k '启动虚拟机' -s 'failed'
```
- 关键字
- 章节
- CPU不足
- 内存不足
- 内部异常
- 序列号过期
- 虚拟机镜像忙
- 存储ID不可访问
- 虚拟磁盘镜像文件不可访问
- 服务不可用，可能由于系统繁忙导致
- 虚拟机正在进行其他操作
- 虚拟机正在开机
- 虚拟机未运行
- 获取集群锁失败
- 获取版本号失败
- 获取CBT信息失败
- 获取快照类型失败
- 不支持的虚拟机兼容性版本号
- 虚拟存储不支持在该主机启动虚拟机
- ARM平台虚拟机需要设置Host CPU
- AMD和海光CPU不支持使用嵌套虚拟化
- 更新网卡设备信息失败：数据处理失败
- 更新网卡设备信息失败：服务不可用
- 更新网卡设备信息失败：存在IP冲突
- 3D虚拟机：显卡切分方式不匹配或显卡资源不足
- 3D虚拟机：当前主机显卡资源不足以开启虚拟机

- 找时间重启内存泄漏的主机才能回收泄漏的内存
