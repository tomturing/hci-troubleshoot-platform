## 实现细节

### 核心改动

将 CI workflow 中的并发触发：

```yaml
for env in dev staging; do
  gh workflow run ... -f target_env=${env}
done
```

改为串行执行：

```yaml
environments=("dev" "staging")
for env in "${environments[@]}"; do
  gh workflow run ... -f target_env=${env}
  sleep 30  # 等待 Git 推送完成
  gh run list ... # 输出状态
done
```

## 验收标准

- [ ] 两个环境都成功同步或 dev 成功 staging 失败时 dev 不受影响
