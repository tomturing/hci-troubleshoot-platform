## 任务清单

### T1: 数据库表设计与创建

- **描述**：创建 terminal_operation 表，包含所有必要字段和索引
- **文件变更**：
  - `database/desired_schema.sql` - 新增表定义
  - `database/migrations/` - 新增迁移脚本（如使用 Alembic）
- **验收标准**：
  - 表创建成功，字段完整
  - 索引生效，查询性能满足要求
- **依赖**：无
- **预计耗时**：1-2h
- **状态**：待开始

### T2: 后端 API 写入接口

- **描述**：实现 POST /api/terminal/operations 接口，接收前端上传的操作记录
- **文件变更**：
  - `backend/api-gateway/app/routes/terminal.py` - 新增路由
  - `backend/api-gateway/app/services/terminal.py` - 新增写入逻辑
  - `backend/api-gateway/app/models/terminal.py` - 新增请求/响应模型
- **验收标准**：
  - 接口正常响应 200
  - 数据正确写入数据库
  - 包含必要的错误处理
- **依赖**：T1
- **预计耗时**：2-3h
- **状态**：待开始

### T3: 后端 API 查询接口

- **描述**：实现 GET /api/terminal/operations 接口，支持按工单/阶段/关键词过滤
- **文件变更**：
  - `backend/api-gateway/app/routes/terminal.py` - 新增路由
  - `backend/api-gateway/app/services/terminal.py` - 新增查询逻辑
- **验收标准**：
  - 按 case_id 过滤正常
  - 搜索功能正常（关键词匹配）
  - 分页正常
- **依赖**：T1
- **预计耗时**：1-2h
- **状态**：待开始

### T4: 前端录制工具函数

- **描述**：创建 terminal-recording.ts 工具模块，包含序号维护、ANSI处理、批量上传逻辑
- **文件变更**：
  - `frontend/customer/src/api/terminal-recording.ts` - 新增文件
- **验收标准**：
  - getNextSeqNumber() 正常递增
  - stripAnsi() 正确剔除 ANSI 码
  - debouncedUpload() 正常批量上传
- **依赖**：T2
- **预计耗时**：2-3h
- **状态**：待开始

### T5: TerminalPanel.vue 录制拦截

- **描述**：在 executeCommand() 和 writeTerminal() 中植入录制逻辑
- **文件变更**：
  - `frontend/customer/src/components/TerminalPanel.vue` - 修改
- **验收标准**：
  - 命令执行后 input 记录写入
  - 输出回显后 output 记录写入
  - 不影响终端实时交互体验
- **依赖**：T4
- **预计耗时**：2-3h
- **状态**：待开始

### T6: TerminalReplay.vue 回放组件

- **描述**：创建终端回放组件，支持时间轴播放和列表查看
- **文件变更**：
  - `frontend/customer/src/components/TerminalReplay.vue` - 新增文件
- **验收标准**：
  - 时间轴控制（播放/暂停/速度）正常
  - xterm.js 正确渲染终端输出
  - 操作详情卡片正常显示
  - 复制/引用功能正常
- **依赖**：T3
- **预计耗时**：4-5h
- **状态**：待开始
