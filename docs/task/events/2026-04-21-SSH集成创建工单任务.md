## 任务清单

### T1: 创建 CaseCreateDialog.vue 组件

- **描述**: 新建工单创建弹框组件，集成基本信息输入、SSH 连接信息输入、Bridge 状态检测
- **文件变更**:
  - 新增 `frontend/customer/src/components/CaseCreateDialog.vue`
- **验收标准**:
  - 弹框包含：标题、描述、AI 助手选择器、SSH 信息输入框
  - 弹框弹出时自动检测 Bridge 状态
  - 按钮排列：连接并创建（蓝）→ 取消（白）→ 无 SSH 创建（灰）
  - 无 SSH 按钮悬浮显示告警提示
- **依赖**: 无
- **预计耗时**: 4h
- **状态**: 待开始

### T2: 改造 chat.ts Store

- **描述**: 新增 SSH 连接流程相关状态和方法，包括 Bridge 检测、SSH 连接、acli 检查、环境采集
- **文件变更**:
  - 修改 `frontend/customer/src/stores/chat.ts`
- **验收标准**:
  - 新增状态：sshConfig、bridgeStatus、sshConnectionPhase、sshErrorInfo、acliAvailable
  - 新增方法：connectSSHAndCreateCase()、checkAcliAvailable()、collectEnvironment()
  - SSH 信息自动填充（localStorage）
  - 复用现有 checkBridgeRunning 函数
- **依赖**: T1（Store 方法被组件调用）
- **预计耗时**: 3h
- **状态**: 待开始

### T3: 实现 acli 检查和采集命令执行

- **描述**: SSH 连接成功后执行 acli 检查，可用则执行三个采集命令，解析输出并提交到 Environment API
- **文件变更**:
  - 修改 `frontend/customer/src/stores/chat.ts`（采集逻辑）
  - 修改 `frontend/customer/src/components/TerminalPanel.vue`（命令执行入口）
- **验收标准**:
  - 执行 `acli` 检查工具是否安装
  - 执行 `acli platform info get` 获取集群信息
  - 执行 `acli --formatter json alert list` 获取告警
  - 执行 `acli --formatter json task list` 获取任务
  - 输出解析正确，提交到 Environment API
  - 采集失败时提示并允许跳过
- **依赖**: T2
- **预计耗时**: 3h
- **状态**: 待开始

### T4: 创建 EnvironmentSummary.vue 展示组件

- **描述**: 创建环境数据折叠卡片组件，用于在对话界面展示采集结果
- **文件变更**:
  - 新增 `frontend/customer/src/components/EnvironmentSummary.vue`
- **验收标准**:
  - 折叠状态：显示集群版本、告警数量、任务状态摘要
  - 展开状态：显示详细信息（集群、告警列表、任务列表）
  - 提供「刷新数据」「发送到助手」按钮
  - 响应式样式适配
- **依赖**: T3（依赖采集数据结构）
- **预计耗时**: 2h
- **状态**: 待开始

### T5: 集成到 App.vue 和 ChatWindow.vue

- **描述**: 将 CaseCreateDialog 和 EnvironmentSummary 集成到现有界面，替换原有工单创建逻辑
- **文件变更**:
  - 修改 `frontend/customer/src/App.vue`
  - 修改 `frontend/customer/src/components/ChatWindow.vue`
- **验收标准**:
  - 点击「新建工单」打开 CaseCreateDialog
  - 创建成功后显示 EnvironmentSummary 卡片
  - 无 SSH 创建的工单不显示环境数据卡片，显示提示信息
  - 终端面板在 SSH 创建成功后自动打开
- **依赖**: T1, T4
- **预计耗时**: 2h
- **状态**: 待开始

### T6: 错误处理和边界场景

- **描述**: 实现 Bridge 未运行、SSH 连接失败、acli 未安装等错误场景的 UI 处理
- **文件变更**:
  - 修改 `frontend/customer/src/components/CaseCreateDialog.vue`
  - 修改 `frontend/customer/src/stores/chat.ts`
- **验收标准**:
  - Bridge 未运行：SSH 连接按钮禁用，显示下载提示，无 SSH 按钮可用
  - SSH 连接失败：显示错误信息和检查项，允许重新输入并重试
  - acli 未安装：提示并允许跳过，继续创建工单
  - 无 SSH 创建：弹框确认提示，列出失去的能力
- **依赖**: T1, T2
- **预计耗时**: 2h
- **状态**: 待开始

### T7: 测试验证和文档更新

- **描述**: 执行完整流程测试，更新客户端设计文档变更历史
- **文件变更**:
  - 修改 `docs/solution/custom-ui/客户端设计.md`（变更历史更新）
- **验收标准**:
  - Bridge 正常时创建工单（有 SSH）流程完整
  - Bridge 未运行时创建工单（无 SSH）流程完整
  - SSH 连接失败场景正确处理
  - acli 未安装场景正确处理
  - 环境数据卡片交互正常
  - 文档变更历史更新
- **依赖**: T5, T6
- **预计耗时**: 2h
- **状态**: 待开始
