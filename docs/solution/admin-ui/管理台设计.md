## 1. 职责边界

| 职责 | 说明 |
|------|------|
| 工单管理 | 分页查询、筛选、导出全部工单；按状态 / 分类 / 助手类型搜索 |
| 用户管理 | 客户列表、操作员账号管理 |
| AI 助手评估看板 | 各助手 composite_score 趋势、差 case 预警、助手对比 |
| 系统监控入口 | 嵌入 Grafana iframe（trace↔log 双向钻取） |
| 知识库管理 | 文档摄入触发、重建索引、分类树维护 |
| Prompt 管理 | system_prompt 模板版本化管理、激活切换 |

**不负责**：对话交互（由 custom-ui 负责）、Pod 调度（Scheduler Service）。

## 2. 页面结构

### 2.1 侧边栏菜单顺序

按业务优先级从高到低排列：

```
Admin UI
├── 仪表盘          ← 高频入口，系统概览
├── 工单管理        ← 核心业务
├── 分类基线        ← 核心配置，诊断分类管理
├── KBD 审核        ← 知识管理（knowledge_atoms 已废弃，知识审核页面已移除）
├── 用户管理        ← 低频运维
└── 系统监控        ← 低频运维，Grafana 嵌入
```

### 2.2 页面功能规划

```
Admin UI
├── 工单管理
│   ├── 工单列表（分页 + 高级筛选）
│   ├── 工单详情（消息历史 + 审计日志）
│   └── 数据导出（CSV / Excel）
├── 助手评估
│   ├── 质量看板（composite_score 趋势折线图）
│   ├── 助手对比（OpenClaw vs NaboBot）
│   └── 差 case 列表（score < 3 或 close_reason=abandon）
├── 系统监控
│   └── Grafana 嵌入（iframe, /d/hci-overview）
├── 知识库管理
│   ├── 文档列表（分类树 + 搜索）
│   ├── 摄入任务（上传 / URL / 批量）
│   └── 重建索引
└── 系统配置
    ├── Prompt 模板管理（版本列表 + 激活切换）
    ├── 工具定义管理（tool_definition 表）
    └── 用户 / 客户管理
```
